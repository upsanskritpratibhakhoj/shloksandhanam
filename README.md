## Getting Started

**Prerequisites:** Node.js (v16 or higher)

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
4. Open your browser and navigate to `http://localhost:5173`

## How to Use

1. Click on "Explore Now" button
2. Type the beginning of a shloka in Devanagari script
3. Browse through matching results in the dropdown
4. Click on any shloka to view its complete details and next character
